# Tip Prediction ML Model 🧠💰

This project uses **multiple linear regression** to predict tips based on restaurant bill, group size, and smoker status.

## 🚀 Features Used
- total_bill
- size
- smoker (encoded)

## 📊 Evaluation Metrics
- MAE: 0.6674
- RMSE: 0.8286
- R² Score: 0.4507

## 🛠️ Tech Stack
- Python
- Pandas, Scikit-learn, Seaborn
- Jupyter Notebook

## 📁 Dataset
- Tips dataset from Seaborn
